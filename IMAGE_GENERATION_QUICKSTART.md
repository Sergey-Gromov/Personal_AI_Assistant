# Быстрый старт: Генерация изображений

## 🎨 Новая возможность!

Ваш бот теперь умеет **автоматически генерировать изображения** с помощью DALL-E 3!

## 🚀 Как использовать

### Способ 1: Автоматическое определение (рекомендуется)

Просто напишите боту, что вы хотите нарисовать:

```
👤 Пользователь: Нарисуй кота в космосе
🤖 Бот: 🎨 Изображение создано! [отправляет картинку]

👤 Пользователь: Создай изображение футуристического города
🤖 Бот: 🎨 Изображение создано! [отправляет картинку]

👤 Пользователь: Сгенерируй закат на пляже
🤖 Бот: 🎨 Изображение создано! [отправляет картинку]
```

### Способ 2: Команда /image

```
/image красный спортивный автомобиль на горной дороге
```

## ⚡ Ключевые слова

Бот понимает запросы со словами:

- **Нарисуй** / draw
- **Создай изображение** / create image
- **Сгенерируй картинку** / generate picture
- **Покажи как выглядит** / show me what
- **Визуализируй** / visualize

## 🔧 Настройка

### 1. Установите новые зависимости

```bash
pip install -r requirements.txt
```

Или установите напрямую:

```bash
pip install aiohttp>=3.9.0
```

### 2. Проверьте API ключ

Убедитесь, что в `.env` файле указан `OPENAI_API_KEY`:

```env
OPENAI_API_KEY=sk-...
```

### 3. Проверьте баланс OpenAI

⚠️ **Важно:** Генерация изображений стоит денег!

- Standard 1024x1024: **$0.040** за изображение
- HD 1024x1024: **$0.080** за изображение

Проверьте баланс на [platform.openai.com](https://platform.openai.com)

### 4. Запустите бота

```bash
python main.py
```

## 🤖 Как это работает

1. **Вы пишете запрос** → "Нарисуй кота в космосе"

2. **ИИ анализирует** → Определяет, что это запрос на генерацию изображения

3. **Улучшение промпта** → GPT улучшает описание для лучших результатов

4. **DALL-E 3 генерирует** → Создает изображение высокого качества

5. **Бот отправляет** → Вы получаете картинку с подписью

## 🎯 Советы для лучших результатов

1. ✅ **Будьте конкретны**: "красный спортивный автомобиль" лучше, чем "машина"
2. ✅ **Указывайте стиль**: "в стиле акварели", "фотореалистично"
3. ✅ **Добавьте детали**: освещение, настроение, цвета
4. ✅ **Используйте референсы**: "в стиле Ван Гога", "как в фильме Blade Runner"

## ⚙️ Настройка параметров

В файле `config.py` можно изменить параметры по умолчанию:

```python
# Размер изображения
DALLE_DEFAULT_SIZE = "1024x1024"  # или "1024x1792", "1792x1024"

# Качество
DALLE_DEFAULT_QUALITY = "standard"  # или "hd"

# Стиль
DALLE_DEFAULT_STYLE = "vivid"  # или "natural"
```

## 🚨 Возможные проблемы

### "Quota exceeded"

**Проблема:** Закончились деньги на счете OpenAI

**Решение:** Пополните баланс на platform.openai.com

### "Content policy violation"

**Проблема:** Запрос нарушает правила OpenAI

**Решение:** Перефразируйте запрос, избегайте запрещенного контента

### Изображение не такое как ожидали

**Решение:** Добавьте больше деталей в описание или попробуйте другие слова
