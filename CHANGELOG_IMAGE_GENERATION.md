- ✅ `/image <описание>` - прямая генерация изображения

### 🎯 Как использовать

#### Для пользователей
```
Просто напишите:
- "Нарисуй кота в космосе"
- "Создай изображение заката"
- "Сгенерируй футуристический город"

Или используйте команду:
/image красное яблоко на столе
```

#### Для разработчиков
```python
from services.image_generation import generate_image

# Генерация изображения
result = await generate_image(
    prompt="A cat in space",
    size="1024x1024",
    quality="standard",
    style="vivid"
)

print(result['image_path'])
print(result['revised_prompt'])
```

### 💰 Стоимость

Генерация изображений через OpenAI API стоит:
- **Standard 1024x1024**: $0.040 за изображение
- **Standard 1024x1792/1792x1024**: $0.080 за изображение
- **HD 1024x1024**: $0.080 за изображение
- **HD 1024x1792/1792x1024**: $0.120 за изображение

⚠️ **Внимание**: Убедитесь, что на балансе OpenAI достаточно средств!

### 🚀 Миграция

#### Шаг 1: Установите зависимости
```bash
pip install aiohttp>=3.9.0
```

Или обновите все зависимости:
```bash
pip install -r requirements.txt
```

#### Шаг 2: Проверьте API ключ
Убедитесь что в `.env` файле есть:
```env
OPENAI_API_KEY=sk-...
```

#### Шаг 3: Проверьте баланс
Зайдите на [platform.openai.com](https://platform.openai.com) и проверьте баланс

#### Шаг 4: Запустите бота
```bash
python main.py
```

#### Шаг 5: Протестируйте
Отправьте боту: "Нарисуй кота"

### ⚙️ Настройка
