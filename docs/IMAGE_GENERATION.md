# Генерация изображений с помощью DALL-E

## Обзор

Бот теперь поддерживает автоматическую генерацию изображений с помощью DALL-E 3! ИИ автоматически определяет, когда пользователь хочет создать изображение, и генерирует его.

## Как использовать

### Автоматическая генерация

Просто напишите боту сообщение с запросом на создание изображения. ИИ автоматически определит ваше намерение и сгенерирует картинку.

**Примеры запросов:**

- "Нарисуй кота в космосе"
- "Создай изображение футуристического города"
- "Сгенерируй картинку заката на море"
- "Покажи как выглядит робот-повар"
- "Визуализируй дракона в горах"

### Команда /image

Вы также можете использовать прямую команду для генерации:

```
/image <описание изображения>
```

**Примеры:**

```
/image красный спортивный автомобиль на горной дороге
/image абстрактная картина в стиле Пикассо
```

## Ключевые слова для определения

Бот распознает следующие ключевые слова (на русском и английском):

- нарисуй / draw
- сгенерируй изображение / generate image
- создай картинку / create picture
- сделай изображение / make image
- покажи как выглядит / show me what
- визуализируй / visualize

## Технические детали

### Параметры генерации

По умолчанию используются следующие параметры:

- **Модель:** DALL-E 3
- **Размер:** 1024x1024
- **Качество:** standard
- **Стиль:** vivid (яркий)

### Процесс генерации

1. **Определение намерения:** ИИ анализирует текст и определяет, нужна ли генерация изображения
2. **Улучшение промпта:** Запрос автоматически улучшается для лучших результатов
3. **Генерация:** DALL-E 3 создает изображение
4. **Отправка:** Изображение отправляется пользователю с подписью (улучшенный промпт)

### Обработка ошибок

Бот обрабатывает следующие типы ошибок:

- **Превышение квоты API:** Сообщение о необходимости проверить баланс OpenAI
- **Нарушение политики контента:** Запрос не соответствует правилам OpenAI
- **Другие ошибки:** Предложение попробовать еще раз или перефразировать

## Настройка

### Конфигурация в config.py

```python
# DALL-E Configuration
DALLE_MODEL = "dall-e-3"
DALLE_DEFAULT_SIZE = "1024x1024"  # Options: 1024x1024, 1024x1792, 1792x1024
DALLE_DEFAULT_QUALITY = "standard"  # Options: standard, hd
DALLE_DEFAULT_STYLE = "vivid"  # Options: vivid, natural
```

### Доступные размеры

- `1024x1024` - квадратное изображение
- `1024x1792` - вертикальное (портрет)
- `1792x1024` - горизонтальное (ландшафт)

### Качество

- `standard` - стандартное качество (быстрее и дешевле)
- `hd` - высокое качество (медленнее и дороже)

### Стиль

- `vivid` - яркий, гипер-реалистичный (по умолчанию)
- `natural` - более естественный, менее драматичный

## Стоимость

Генерация изображений с DALL-E 3:

- **Standard 1024x1024:** $0.040 за изображение
- **Standard 1024x1792 или 1792x1024:** $0.080 за изображение
- **HD 1024x1024:** $0.080 за изображение
- **HD 1024x1792 или 1792x1024:** $0.120 за изображение

⚠️ **Важно:** Убедитесь, что у вас достаточно средств на балансе OpenAI.

## Хранение изображений

Сгенерированные изображения временно сохраняются в директории `data/generated_images/` и автоматически удаляются после отправки пользователю.

## Политика контента

DALL-E 3 имеет строгую политику контента. Нельзя генерировать изображения:

- С насилием или жестокостью
- С сексуальным или эротическим содержанием
- С изображением известных людей
- С защищенными авторским правом персонажами
- С ненавистническим контентом

При нарушении политики бот сообщит об ошибке.

## Советы для лучших результатов

1. **Будьте конкретны:** Чем детальнее описание, тем лучше результат
2. **Указывайте стиль:** "в стиле акварели", "фотореалистично", "мультяшный стиль"
3. **Упоминайте освещение:** "при закатном свете", "яркий полдень", "ночное освещение"
4. **Добавьте атмосферу:** "драматичный", "спокойный", "энергичный"
5. **Используйте референсы:** "как на картинах Моне", "в стиле cyberpunk"

## Troubleshooting

### Ошибка: "Quota exceeded"

**Решение:** Проверьте баланс на платформе OpenAI

### Ошибка: "Content policy violation"

**Решение:** Перефразируйте запрос, убедитесь что он соответствует политике контента

### Изображение не соответствует ожиданиям

**Решение:** Попробуйте добавить больше деталей в описание или использовать другие слова

### Долгая генерация

**Решение:** DALL-E 3 может занять 10-30 секунд на генерацию. Это нормально.

## API Reference

### `detect_image_generation_intent(text, conversation_history)`

Определяет, нужна ли генерация изображения.

**Возвращает:**
```python
{
    "needs_generation": bool,
    "prompt": str,  # улучшенный промпт
    "confidence": float  # 0.0-1.0
}
```

### `generate_image(prompt, size, quality, style)`

Генерирует изображение с DALL-E 3.

**Параметры:**
- `prompt` (str): Описание изображения
- `size` (str): Размер изображения
- `quality` (str): Качество (standard/hd)
- `style` (str): Стиль (vivid/natural)

**Возвращает:**
```python
{
    "image_path": Path,
    "revised_prompt": str,
    "url": str,
    "original_prompt": str
}
```
